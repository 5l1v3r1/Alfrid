Alfrid
======

A WebGL tool set

### Default Vertex Attributes
- aVertexPosition
- aTextureCoord
- aNormal

### Default Matrix Uniforms
- uModelMatrix
- uViewMatrix
- uProjectionMatrix
- uNormalMatrix
- uModelViewMatrixInverse


### Installation 

```
npm install;
```

### Development 

```
npm run dev;
```


### Build 

```
npm run build;
```
